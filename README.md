# Investigación de Mercados – UNP
Aplicación educativa en HTML convertida en PWA para uso de alumnos en dispositivos móviles.

## 📱 Instalación
1. Abrir el enlace en el navegador del celular.
2. Seleccionar “Agregar a pantalla principal”.
3. La aplicación se instalará con el ícono y nombre definidos.

## 📂 Archivos principales
- `index.html`: Programa principal.
- `manifest.json`: Configuración de la aplicación.
- `icono_investigacion.png`: Ícono de la app.

Proyecto académico desarrollado para fortalecer el aprendizaje de Investigación de Mercados en la UNP.
